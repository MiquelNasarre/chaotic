# Contributing to Chaotic

## Quick Start

1. Fork the repository
2. Clone your fork
3. Open `chaotic.sln` in Visual Studio 2022
4. Build and test with `chaotic_demo()`
5. Make changes and submit a PR

## Requirements

- Windows 10/11
- Visual Studio 2022 with C++ workload
- Windows SDK

## Code Style

- Use tabs (4 spaces)
- Classes: `PascalCase`
- Functions/variables: `camelCase`
- Opening braces on same line
